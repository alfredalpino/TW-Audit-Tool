# Weekly SEO Optimization Loop

This workflow converts SEO activity into a repeatable operating loop.

## Weekly Runbook (90 minutes)

## 1) Measure (20 min)

- Pull Search Console for last 7 days vs prior 7 days.
- Check:
  - non-branded impressions
  - non-branded clicks
  - average position for priority queries
  - landing pages with rising impressions but weak CTR
- Pull GA4 for organic conversions:
  - discovery-call clicks
  - booking CTA clicks
  - contact/lead submissions (if applicable)

## 2) Diagnose (20 min)

- Identify 3 underperforming pages:
  - high impressions, low CTR
  - stable traffic, low conversion
  - ranking drift for priority terms
- For each page, note likely blocker:
  - title/H1 mismatch
  - intent mismatch
  - weak internal links
  - thin section depth

## 3) Implement (35 min)

- Ship exactly 3 updates each week:
  1. title + meta update on one page
  2. H2/content improvement on one page
  3. internal links added from blog/supporting pages to one service/commercial page

## 4) Publish and Log (15 min)

- Submit updated URLs in Search Console inspection.
- Update weekly change log:
  - date
  - page URL
  - change type
  - target query/theme
  - expected outcome

## KPI Targets (track trendline)

- Referring domains growth (monthly)
- Non-branded clicks growth (weekly)
- Organic conversion rate (weekly)
- Discovery-call volume from organic (weekly)
